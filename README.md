# Kiva Lite 📱

Kiva Lite is a short-video social media application inspired by modern video-sharing platforms. It allows users to upload, watch, like, comment on, and share short videos.

## ✨ Features

- 🔐 User Authentication
- 🎥 Upload Short Videos
- ▶️ Video Feed
- ❤️ Like Videos
- 💬 Comment on Videos
- 👤 User Profiles
- 🔍 Search Users & Videos
- 🔔 Notifications
- 📤 Share Videos

## 🛠️ Built With

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio
- Visual Studio Code (Optional)
- Firebase Project

### Installation

```bash
git clone https://github.com/yourusername/kiva-lite.git
```

```bash
cd kiva-lite
```

```bash
flutter pub get
```

```bash
flutter run
```
